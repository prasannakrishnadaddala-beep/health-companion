1 Install Google Libraries

Open Command Prompt and run:

pip install google-auth google-auth-oauthlib google-api-python-client

Then restart the app (run.bat) and refresh this page.

2 Create Google Cloud Credentials
  1. Go to console.cloud.google.com
  2. Create a new project → name it "Health Companion"
  3. Go to APIs & Services → Library
  4. Search for "Fitness API" → Click Enable
  5. Go to APIs & Services → Credentials
  6. Click Create Credentials → OAuth client ID
  7. Application type: Desktop app
  8. Click Download JSON
  9. Rename the file to google_credentials.json
  10. Place it in your health_companion folder
⚠️ Also go to OAuth consent screen → Add your Gmail as a Test User
3 Connect Your Google Account

Make sure your Fastrack Active app is synced on your phone first, then click the button below. A browser window will open — sign in with the same Google account linked to Google Fit.

✅ Connected! Go back to the Health Companion app and click the Sync button on the Vitals page.
4 Fastrack Watch Setup
  1. Open Fastrack Active app on your wife's Android phone
  2. Go to Settings → Connected Apps
  3. Enable Google Fit sync
  4. Wear the watch and let it sync (takes ~5 mins)
  5. Come back here and click Connect Google Fit
💡 Fastrack Reflex Ultra / Opus series sync heart rate every minute while active. O2 and sleep data sync when you open the app.
